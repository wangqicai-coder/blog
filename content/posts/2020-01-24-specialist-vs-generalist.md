## 引子：两种成功路径

**Steve Jobs**：典型的通才
- 懂技术（但不是最顶尖的工程师）
- 懂设计（但不是专业设计师）  
- 懂商业（但不是MBA出身）
- **跨界整合能力** → 创造了iPhone

**Linus Torvalds**：典型的专家
- Linux内核开发者
- 50岁仍在写代码
- **极致专精** → 改变了操作系统世界

两种路径，都能成功。

**问题是：你应该选哪条？**

今天我们探讨：
- 专业化vs通才化的权衡
- T型、π型、梳子型人才
- 如何根据你的情况选择

## 核心概念：三种人才模型

### 1. I型人才（专家）

```
    │
    │  ← 深度
    │
    ▼
```

**特征：**
- 在单一领域极度专精
- 10年+的专注积累
- 成为该领域的前1%

**优势：**
- 不可替代性强
- 在专业领域有定价权
- 复利效应（越深入越有优势）

**劣势：**
- 视野局限
- 行业变化时脆弱
- 难以跨界创新

**适合：**
- 技术密集型领域（如外科医生、飞行员）
- 需要长期积累的领域（如法律、会计）

### 2. - 型人才（通才）

```
──────────  ← 广度
```

**特征：**
- 什么都懂一点
- 但都不精通

**优势：**
- 视野开阔
- 适应性强
- 容易找到机会

**劣势：**
- 没有核心竞争力
- 容易被替代
- 难以建立壁垒

**适合：**
- 早期探索阶段
- 快速变化的环境
- **但不适合作为长期策略**

### 3. T型人才（理想模型）

```
──────────  ← 广度（跨学科思维）
    │
    │        ← 深度（专业领域）
    ▼
```

**特征：**
- **横向**：跨学科的思维模型和知识
- **纵向**：某个领域的深度专业能力

**优势：**
- 专业能力（竞争力）
- 跨界视野（创新力）
- **两者结合 = 非线性优势**

**这是Charlie Munger推崇的模型。**
