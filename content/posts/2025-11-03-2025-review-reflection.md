### 3月：风险管理

**主题**：保护本金=第一要务

**核心文章**：
- 3/3：2%法则的数学基础
- 3/5：止损的心理学
- 3/9：破产风险计算

**关键收获**：

```

**成长自评**：

**如果你答"是"≥2个** → 你有风险意识 ✓

**核心文章**：

- 6/6：群体狂热识别
- 6/9：周期中的心理陷阱

**关键收获**：

```

**成长自评**：

### 7月：行为金融学

**主题**：系统性非理性

**核心文章**：
- 7/2：前景理论
- 7/5：心理账户
- 7/8：羊群效应

**关键收获**：
```
人类决策规律：
- 前景理论：亏损痛苦>盈利快乐（2.5倍）

→ 警惕自己也是"人"
```

**成长自评**：

### 8月：组合管理

**主题**：分散的真谛

**核心文章**：
- 8/3：核心-卫星策略
- 8/7：多策略组合
- 8/11：再平衡的挑战

**关键收获**：
```
真正分散≠买10只股票：

核心-卫星：
- 核心70%：稳健（指数+龙头+债券）

```

**成长自评**：

- [ ] 我的策略相关性<0.3吗？
- [ ] 我定期再平衡（季度/偏离>10%）吗？

**如果你答"是"≥2个** → 你有组合思维 ✓

### 9月：心理韧性

**主题**：从挫折中恢复

**核心文章**：
- 9/2：心理韧性培养
- 9/6：应对连续亏损
- 9/10：避免过度交易

**关键收获**：
```
韧性=核心竞争力：
- 不是"不亏损"，是"亏损后快速恢复"
- Seligman 3P模型：
  低韧性→永久/普遍/个人归因
  高韧性→暂时/具体/外部归因

ABCDE模型：
A（事件）→B（信念）→C（后果）
→D（辩驳）→E（新赋能）
```

**成长自评**：
- [ ] 我从连续亏损中恢复时间<24小时吗？
- [ ] 我使用ABCDE日志管理情绪吗？
- [ ] 我有稳定的支持系统（社群/导师）吗？

**如果你答"是"≥2个** → 你的韧性在增强 ✓

## 结语：2025→2026

**核心收获**：

2025年，你学到的不仅是交易技巧，更是：
- **认知升级**：识别偏差→系统思考
- **情绪管理**：ABCDE模型→韧性培养
- **长期主义**：10年视角→复利思维

**最重要的洞察**：

> 交易心理学≠让你每笔交易都赚钱
> 而是让你：
> - 亏损时不崩溃
> - 盈利时不冲动
> - 长期坚持系统
> - **最终，时间和复利站在你这边**
