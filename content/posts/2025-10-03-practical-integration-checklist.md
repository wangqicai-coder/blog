## 第一部分：决策清单框架

### 三层架构

**Layer 1：战前准备**（长期）
- 系统设计
- 心理建设

- 系统优化
- 时间：交易后/周末

### 每日例行程序

**晨间仪式**（20分钟）

```
6:00-6:10 心理准备
- [ ] 冥想/深呼吸5分钟
- [ ] 回顾长期目标（10年视角）
- [ ] 今日意图设定："今天我要___________"

6:10-6:20 信息输入
- [ ] 检查重要新闻（限10分钟，避免过度）
- [ ] 市场概况（指数、关键品种）
- [ ] 避免：社交媒体、情绪化内容

6:20-6:30 系统检查
- [ ] 查看持仓（如有）
- [ ] 今日是否有系统信号？
- [ ] 预设止损/止盈提醒
```

**盘中决策流程**（每次交易前）

**阶段1：信号确认**（2分钟）
```
[ ] 这是100%符合系统的信号吗？
    是 → 继续
    否/不确定 → 停止，不交易

[ ] 信号质量评分（1-5星）：___
    5星：所有条件完美
    4星：多数条件符合
    3星及以下 → 放弃

[ ] 我是否因情绪想交易？
    - [ ] 无聊？
    - [ ] FOMO（怕错过）？
    - [ ] 报复性（想搏回来）？
    任何一项是 → 停止24小时
```

**参考**：
- 9/10：避免过度交易（工具2：交易前检查清单）

**阶段2：风险计算**（3分钟）

```
[ ] 入场价：$_______
[ ] 止损价：$_______
[ ] 止损距离：____%
[ ] 仓位计算（2%风险）：
    允许亏损：账户×2% = $_______
    仓位 = 允许亏损 / 止损距离 = ______股

[ ] 仓位验证：
    - 单笔≤总资产5%？___
    - 不超过配额（9/10交易券）？___
    - 如否 → 调整仓位或放弃
```

**参考**：
- 3/3-3/5：风险管理系列

## 第五部分：特殊情况应对清单

### 情况1：连续亏损（≥3次）

**立即行动**：
```
[ ] 停止交易24-48小时
[ ] 查看历史数据：
    我的系统最长连亏：___次
    当前连亏：___次
    是否在正常范围？___

[ ] 使用9/6工具：数据驱动信心
    统计分析 → 这是概率事件

[ ] ABCDE日志：
    A：连亏3次
    B："我的系统失效了"
    D：系统胜率55%，连亏3次概率9%，正常
    E："这是正常波动，坚持"

[ ] 如果连亏>历史最长：
    → 深度回测
    → 考虑市场环境变化
    → 咨询导师
```

**参考**：9/6应对连续亏损完整方法

### 情况2：大幅盈利后（单日>10%）

**警惕过度自信**：
```
[ ] 庆祝（5分钟）
[ ] 然后提醒自己：
    "这是运气+技能，不是纯技能"
    "下一笔独立于这一笔"

[ ] 强制冷却期：48小时不交易
    （避免：一把大赚→冲动加仓→归零）

[ ] 提取部分利润（可选）：
    提取___%作为奖励
    剩余继续复利

[ ] 回归系统：
    "下一笔仍然按系统信号"
```

**参考**：1/9-1/15情绪管理（特别是1/12贪婪管理）

### 情况3：黑天鹅事件（市场暴跌>20%）

**应急清单**：
```
[ ] 第1小时：不做任何决策
    深呼吸，远离电脑

[ ] 第2-4小时：评估损失
    当前回撤：-____%
    是否触及止损？___
    账户安全吗？___

[ ] 第1天：执行止损（如触及）
    机械执行，不犹豫
    （记住：3/5文章，止损=保护）

[ ] 第2-3天：冷静分析
    这是系统性风险还是个股？
    我的对冲策略生效了吗？
    （参考8/7多策略）

[ ] 第1周：重新规划
    损失已定，接受
    重建信心（9/2韧性工具）
    调整仓位（降低风险）

[ ] 不要：
    - [ ] 底部抄底（情绪化）
    - [ ] 加大杠杆"搏回来"
    - [ ] 放弃系统
```

**参考**：
- 6/2-6/9：市场周期系列
- 9/2：心理韧性

### 情况4：想放弃系统

**检查清单**：
```
[ ] 我的系统运行多久了？
    <6个月 → 时间不够，继续
    6-12个月 → 评估，但谨慎
    >12个月 → 可以考虑调整

[ ] 系统失效的证据：
    - [ ] 连续3个季度低于期望值
    - [ ] 市场结构性变化（有数据支持）
    - [ ] 回测显示失效

    如果<2项 → 坚持系统

[ ] 我是否在情绪化？
    最近连亏？___
    看到别人赚钱？___
    FOMO？___

    如果是 → 这是情绪，非理性

[ ] 10/10/10法则（9/14）：
    10分钟后："我很frustrated"
    10个月后："我会后悔换系统"
    10年后："这不重要"

    结论：___
```

**如果真的需要调整**：
```
不要全部推倒重来
而是：
1. 微调（10-20%改变）
2. 重新回测
3. 小仓位测试3个月
4. 评估后决定
```

### 案例：张华的完整一天

**背景**：
- 交易3年
- 使用趋势跟随系统
- 账户$200,000

**6:00 晨间仪式**
```
✓ 冥想5分钟（平静心态）
✓ 回顾10年目标（贴在墙上）
✓ 今日意图："无论盈亏，执行系统"
✓ 快速看新闻（10分钟，无重大事件）
```

**9:30 开盘**
```
检查系统信号：
- 持仓A：无变化，继续持有
- 新信号？检查筛选器
  → 发现品种B突破20日新高+成交量确认
  → 5星信号✓
```

**9:35 决策流程**
```
阶段1：信号确认
✓ 100%符合系统
✓ 5星信号
✓ 情绪检查：无FOMO，冷静

阶段2：风险计算
入场：$50
止损：$48（-4%）
账户：$200,000
允许亏损：$200k × 2% = $4,000
仓位：$4,000 / ($50-$48) = 2,000股
✓ 仓位=2000×$50=$100k（50%账户，需调整）
→ 调整至1000股（25%账户，OK）

阶段3：情绪自检
焦虑：3/10 ✓
兴奋：5/10 ✓
愤怒：0/10 ✓
→ 情绪OK

阶段4：执行
✓ 买入1000股@$50
✓ 设置止损$48
✓ 记录日志
✓ 关闭软件
```

**下午3:00 品种B触及止损**
```
系统自动执行：
- 卖出1000股@$48
- 亏损：-$2,000（-1%账户）

情绪反应：
- 失望（正常）
- 想法："是不是信号错了？"

应对：
✓ 深呼吸
✓ 提醒自己："胜率55%，这是45%的正常亏损"
✓ 不做报复性交易
✓ 遵守48小时冷却期（9/10规则）
```

**20:00 晚间复盘**
```
交易回顾：
✓ 1笔，符合系统
✓ 执行质量：9/10
✓ 亏损-1%，在风控内

情绪日志：
- 波动：6/10（失望但可控）
- 挑战：接受亏损
- 应对：深呼吸+理性分析 ✓

ABCDE日志：
A：止损-$2000
B："我是不是做错了？"
D：系统胜率55%，亏损是正常，我执行完美
E："这是好的交易（过程），结果只是运气"

感恩：
1. 系统保护了我（只亏1%）
2. 我执行了纪律
3. 我在成长
```

**周日复盘**（本周3笔交易）
```
业绩：
- 3笔，2胜1负
- 胜率：67%（>系统期望55%）
- 本周：+2.5%
- 系统执行率：100% ✓

心理：
- 最大挑战：周三的止损
- 应对：ABCDE日志有效
- 韧性得分：42/50（vs上周40）

- 读完《海龟交易法则》

- 继续100%执行系统
- 阅读：《原则》
```

### 个性化调整

**清单不是教条**

**例**：
- 不喜欢冥想？→ 改为散步10分钟
- 晚间没时间？→ 改为早晨复盘
