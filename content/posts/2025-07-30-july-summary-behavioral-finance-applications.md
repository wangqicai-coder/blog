## 第一部分：7月知识地图

### 七大认知偏差

**1. 锚定效应**（7/2）：
- 过度依赖第一条信息
- 买入价、历史高点成为心理锚
- 克服：归零思维、隐藏成本价

**2. 禀赋效应**（7/6）：
- 拥有→高估价值
- 止损困难
- 克服：第三方视角、机械止损

**3. 框架效应**（7/10）：
- 表述方式改变决策
- 媒体操纵情绪
- 克服：双向框架、数字化

**4. 可得性偏差**（7/14）：
- 容易想起→高估概率
- 新闻误导决策
- 克服：数据替代记忆、基准率

**5. 后见之明偏差**（7/18）：
- "我早知道"幻觉
- 复盘陷阱
- 克服：事前日志、过程评估

**6. 心理账户**（7/22）：
- 给钱分类，不可替代
- 盈利轻兑、亏损死扛
- 克服：统一账户、重置成本

**7. 社会认同与羊群**（7/26）：
- 从众压力
- 追涨杀跌
- 克服：独立思考、逆向检查

## 第二部分：偏差关联图

### 偏差如何叠加

**案例：2015年散户亏损**

**Phase 1（买入）**：
- **可得性**：新闻"暴涨"→高估涨概率
- **社会认同**："大家都买"→跟随
- **框架效应**：媒体"牛市才开始"（增益框架）
- **结果**：高位买入

**Phase 2（持有）**：
- **锚定**：锚定买入价+历史更高点
- **禀赋**："我的股票"→不舍得卖
- **心理账户**："本金不能亏"→死扛
- **结果**：从小亏拿成大亏

**Phase 3（恐慌）**：
- **可得性**：新闻"股灾"→高估继续跌概率
- **社会认同**："大家都卖"→恐慌抛售
- **结果**：底部割肉

**总结**：7个偏差环环相扣→巨额亏损

### 偏差权重

**不同阶段，不同偏差主导**：

| 阶段 | 主导偏差 | 表现 |
|------|---------|------|
| 入场决策 | 可得性、社会认同、框架 | 追热点 |
| 持仓管理 | 锚定、禀赋、心理账户 | 不止损 |
| 卖出决策 | 后见之明、社会认同 | 底部割肉/错失卖点 |

## 第三部分：偏差克服系统

### 完整决策流程（1-7月整合）

**0. 周期评估**（6月）
- 五维度评分→确定阶段→基础仓位

**1. 机会识别**
- 信号出现

**2. 偏差过滤（新增）**
```
锚定检查：
- [ ] 我是否被买入价/历史价影响？
- [ ] 归零思维：今天第一次看到，会买吗？

可得性检查：
- [ ] 我的判断是基于新闻，还是数据？
- [ ] 基准率验证

框架检查：
- [ ] 我是否被媒体表述影响？
- [ ] 双向框架：增益+损失都考虑了吗？

社会认同检查：
- [ ] 我是否因为"大家都买"？
- [ ] 延迟48小时，独立分析

如果任何一项触发→暂停决策→深度分析
```

**3. 估值分析**
- PE百分位、DCF、相对估值

**4. 概率思维**（1月）
- 胜率、盈亏比、期望值

**5. 情绪检查**（2月）
- 个人情绪状态
- 市场情绪状态

**6. 逆向指标**（6月）
- 是否极端？
- 需要逆向吗？

**7. 二阶思维**（5月）
- "然后呢？"×3

**8. 系统思维**（5月）
- 反馈循环、系统状态

**9. 非对称性**（4月）
- 风险收益比

**10. 谦卑检查**（5月）
- 能力圈内吗？

**11. 仓位决策**
- 周期基础仓位 ± 个股调整

**12. 执行**

**13. 持仓管理（新增偏差防御）**
```
禀赋防御：
- 定期问："第三方会建议我卖吗？"
- 机械止损（不依赖主观）

心理账户防御：
- 只看总资产（不分本金/盈利）
- 百分比思维

锚定防御：
- 隐藏成本价
- 每日重置："今天第一次买入"心态
```

**14. 复盘（新增）**
```
后见之明防御：
- 查阅事前日志（还原当时真实情况）
- 评估过程，非结果
- 避免"我早知道"
```

**15. 长期视角**
- 符合10年目标吗？

## 第四部分：偏差防御工具包

### 工具1：偏差检查清单

**每次重大决策前**：

```
[ ] 锚定：归零思维测试通过
[ ] 禀赋：第三方视角测试通过
[ ] 框架：双向框架分析完成
[ ] 可得性：数据验证（非新闻）
[ ] 后见之明：事前记录日志
[ ] 心理账户：统一账户视角
[ ] 社会认同：独立分析（非跟风）

7项全部✓→可决策
任何一项✗→深度分析
```

### 工具2：决策日志模板

**目的**：对抗后见之明、锚定、可得性

**格式**：
```
日期：2024-XX-XX
交易：XXX

事前记录（关键）：
- 当前信息：
- 不确定性：?/10
- 可能情景：
  * 情景1：...（概率X%）
  * 情景2：...（概率Y%）
- 群体共识：
- 我的独立分析：
- 偏差检查：7项全部✓

决策：
原因：

事后复盘：
- 参考事前日志（不依赖记忆）
- 过程合理性评估
- 学习点
```

### 工具3：账户视图重置

**交易软件设置**：
```
隐藏：
- 成本价
- 盈亏金额
- 盈亏比例

只显示：
- 当前价
- 持仓市值
- 总资产

效果：对抗锚定、禀赋、心理账户
```

### 工具4：独立思考日

**每周日**：
```
规则：
- 不看任何媒体、群聊
- 纯基于数据分析
- 独立决策

对比：
- 上周受媒体影响的决策质量
- 本周独立决策的质量

通常：独立质量更高
```

## 第五部分：综合案例

### 案例：王敏的7月实践

**背景**：已掌握1-6月内容

**7月目标**：识别并克服自身偏差

**Week 1：发现锚定陷阱**

**持仓**：某股$50买入，现$42

**旧反应**：
- "等回$50"（锚定买入价）

**学习后**：
- 归零思维测试："今天第一次看到$42，我会买吗？"
- 分析：基本面恶化，不会买
- **决策：$42卖出**
- 后续：跌至$35
- "锚定让我及时止损"

**Week 2：克服禀赋效应**

**持仓审计**：
- 某劣质股，亏损-15%
- 舍不得卖（禀赋效应）

**第三方视角**：
- "如果朋友持有，我会建议他卖吗？"
- 答案：会（基本面差）
- **决策：卖出**
- "第三方视角帮我清理垃圾"

**Week 3：避开框架陷阱**

**遭遇**：
- 券商报告："XX股目标价$80（当前$50）"（增益框架）

**双向框架分析**：
```
增益：+60%潜力
概率：30%（券商历史准确率）

损失：-25%风险
概率：40%（估值已偏高）

中性（+10%）：30%

期望值：30%×60% + 40%×(-25%) + 30%×10%
       = 18% - 10% + 3% = 11%

风险收益比：25%/60% = 0.42（不佳）
```

**决策**：小仓位买入（15%），而非重仓
- 后续：+8%卖出
- "双向框架让我控制仓位"

**Week 4：抵抗社会认同**

**环境**：
- 某板块全民热炒
- 朋友都在买
- FOMO强烈

**独立思考流程**：
1. 延迟48小时
2. 逆向检查：
   - [x] 全民讨论
   - [x] 媒体一致看好
   - [x] 估值>90%分位
3. 数据验证：
   - 板块平均PE：历史95%分位
   - 融资余额：暴增
4. **决策：不买**

**结果**：
- 2周后该板块-18%
- "独立思考让我避开泡沫"

**7月总结**：
- 收益：+4.1%
- **更重要**：
  - 识别偏差：9次
  - 成功克服：7次（78%）
  - 避免亏损：估计-15%
- "认识偏差=最大进步"

## 结语：偏差即机会

**7月核心智慧**：
- 认识偏差=克服偏差的开始
- 每个偏差都有克服方法
- 系统化偏差防御>依赖意志力

**七大偏差速记**：
1. **锚定**→归零思维
2. **禀赋**→第三方视角
3. **框架**→双向分析
4. **可得性**→数据验证
5. **后见之明**→事前日志
6. **心理账户**→统一视角
7. **社会认同**→独立思考

**实践要点**：
1. 决策前：7项偏差检查清单
2. 决策中：事前日志记录
3. 决策后：过程复盘（非结果）
4. 账户设置：隐藏成本价
