**9:45 分析医药股信号**

陈阳使用10/3文章的"4阶段决策流程"：

**阶段1：信号确认（2分钟）**
```
✓ 突破60日新高？是
✓ 成交量确认？是（放大1.5倍）
✓ 基本面支持？检查→ROE 18%，负债率35%，行业前3
✓ 5星信号？→ 4星（小瑕疵：PE略高）

决定：符合系统，继续评估
```

**阶段2：风险计算（3分钟）**
```
入场价：$45.00
止损价：$43.00（-4.4%）
账户：$1,500,000
允许亏损：$1.5M × 2% = $30,000
仓位：$30,000 / ($45-$43) = 15,000股
总投入：15,000 × $45 = $675,000

验证：
- $675k / $1.5M = 45%（太高）
→ 调整至仓位：6,000股（$270k，18%账户）✓
```

**阶段3：情绪自检（1分钟）**
```
为什么想做这笔交易？
- [ ] FOMO（看到别人赚）？否
- [ ] 无聊？否
- [ ] 信号质量？是✓

当前情绪：
- 焦虑：4/10 ✓
- 兴奋：5/10 ✓
- 冲动：2/10 ✓

→ 情绪OK，可以交易
```

**阶段4：执行（10:00）**
```
✓ 下单：买入6,000股@$45.20（实际成交价）
✓ 设置止损：$43.00（机械执行）
✓ 记录交易日志：
   理由：5日突破+成交量+基本面
   预期：捕捉趋势初期
   风险：-2%账户
✓ 关闭软件（避免盯盘）
```

> "不要追求完美的一天，追求可复制的系统。我的这一天，重复了3000+次（12年），才有今天的结果。Start small, stay consistent."
