## 引言：反着想

**Charlie Munger**：
> "Invert, always invert. Many problems are best solved backwards."

**逆向思维**：
- 不问"如何成功"
- 而问"如何避免失败"

**数学家Jacobi**："Invert, always invert"（反转，永远反转）

---

## 第一部分：逆向思维原理

### 正向vs逆向

**正向**："我如何赚钱？"
- 寻找机会
- 积极行动
- **问题**：机会众多，难以筛选

**逆向**："我如何避免亏钱？"
- 识别陷阱
- 消除风险
- **优势**：亏钱方式有限，更易控制

**Munger实践**：
- "Tell me where I'm going to die, so I never go there."

### 避免失败>追求成功

**原因**：
1. **不对称性**：
   - 亏损50% → 需要+100%恢复
   - 避免大亏 > 抓住大涨

2. **失败模式有限**：
   - 成功路径无限（难以穷尽）
   - 失败模式有限（可以列举）

3. **查理芒格清单**：
   - 不投资不懂的
   - 不买高估值的
   - 不碰管理层有问题的

## 第二部分：交易中的逆向思维

### 反向问题清单

**正向**："这笔交易能赚钱吗？"

**逆向检查清单**：
- [ ] 这会导致破产吗？
- [ ] 这违反我的规则吗？
- [ ] 我是在FOMO吗？
- [ ] 最坏情况我能承受吗？
- [ ] 这是因为贪婪/恐惧的情绪决策吗？

**只有全部No，才考虑执行**

### Munger的"三个筛选器"

**1. 理解筛选**：
```
If 不完全理解
Then 不投资
```

**2. 估值筛选**：
```
If 估值不便宜
Then 不买入
```

**3. 管理层筛选**：
```
If 管理层不诚信
Then 远离
```

**剩下的才是"可投资清单"**

## 第三部分：失败模式分析

### 交易者破产的10种方式

**Munger式思考**："如何确保我破产？"

1. **过度杠杆**（最快方式）
2. **无止损**（慢性死亡）
3. **追涨杀跌**（情绪化）
4. **满仓单只股票**（集中风险）
5. **频繁交易**（成本侵蚀）
6. **听消息炒股**（信息不对称）
7. **不复盘学习**（重复错误）
8. **报复性交易**（情绪失控）
9. **不懂装懂**（能力圈外）
10. **短期暴富心态**（高风险偏好）

**逆向策略**：**避免以上所有行为**

### 负面清单

**Munger方法**：
- 不列"要做什么"
- 列"**绝对不做**什么"

**交易负面清单**：
- ❌ 不用超过1:1杠杆
- ❌ 不买不懂的股票
- ❌ 不在情绪>7/10时交易
- ❌ 不追涨停板
- ❌ 不听小道消息
- ❌ 不在亏损后加仓（摊平）
- ❌ 不违反系统规则

## 第四部分：逆向选股

### 排除法

**步骤**：
1. **全市场**（3000+只股票）
2. **排除不懂的**（剩500只）
3. **排除高估值的**（剩100只）
4. **排除财务有问题的**（剩30只）
5. **排除管理层有瑕疵的**（剩10只）
6. **在剩下的10只中，挑选最佳**

**效率**：
- 直接选最佳（从3000中选）→困难
- 排除不合格（筛选器）→简单

### 反向指标

**问**："什么时候我**不应该**买入？"

**清单**：
- 估值>行业平均30%
- 连续亏损3年
- ROE<10%
- 负债率>70%
- 大股东减持
- 财务造假嫌疑

**应用**：任何一条触发→从池子删除

## 第五部分：逆向时机

### 反向市场情绪

**Munger/Buffett准则**：
- 市场恐慌（VIX>40）→贪婪
- 市场狂热（融资余额创新高）→恐惧

**逆向信号**：
- 出租车司机谈股票→顶部信号
- 亲友问"还能买吗"→危险区
- 媒体头条"牛市来了"→谨慎

**反向操作**：
- 大众买入→我观望/减仓
- 大众恐慌→我加仓

### 反向自己

**问题**："我什么时候最容易犯错？"

**自我分析**：
- 连续盈利后→过度自信→加大仓位
- 连续亏损后→报复心态→冲动交易
- 看到涨停板→FOMO→追高

**逆向规则**：
```
If 连续盈利3笔
Then 强制休息1天（防止过度自信）

If 连续亏损5笔
Then 减半仓位（防止情绪失控）

If 看到涨停板想追
Then 强制等待24小时
```

## 第六部分：案例研究

### 案例：李明的逆向转型

**2020年：正向思维阶段**

**心态**："我要找到10倍股"

**行为**：
- 追逐热点（新能源、芯片、医药）
- 频繁换股（追涨）
- 高估值也买（"未来空间大"）

**结果**：
- 1年交易80次
- 收益：-12%
- "我很努力，为何亏钱？"

**2021年：逆向思维觉醒**

**转变**：读《穷查理宝典》

**新方法**：
- 不问"如何找牛股"
- 而问"**如何避免地雷股**"

**负面清单**：
```
不买的股票：
1. 我不懂的行业
2. PE>50的（无论多热门）
3. 连续亏损的
4. 负债率>60%的
5. 最近1年股东减持>10%的
```

**实践**（2021年）：
- 从3000只筛选至200只（排除不懂）
- 再排除高估值→50只
- 再排除财务问题→15只
- **最终池子**：15只优质股

**选股**：
- 在15只中，挑选5只建仓
- 而非在3000只中盲目寻找

**2021年结果**：
- 交易次数：80次→15次
- 收益：-12%→+18%

**2022-2024年：系统化逆向**

**进一步完善**：

**负面清单扩展**：
```
不做的事：
1. 不用杠杆
2. 不追涨停
3. 不在情绪>7时交易
4. 不听消息（只看财报）
5. 不买ST股票
6. 不摊平亏损（止损后不回补）
7. 不违反系统规则
```

**正面清单（极简）**：
```
只做的事：
1. 在负面清单之外
2. 符合3个标准：
   - 理解
   - 合理估值
   - 优质管理
```

**结果**：
- 3年年化：+17%
- 波动率大幅降低
- 心态："我不需要找到最好的，只需要避开最差的"

**关键洞察**：
- "避免愚蠢比追求聪明更容易"
- "我的成功不是因为找到了金矿，而是避开了地雷"

## 结语：简单但不容易

**Munger的智慧**：
- 逆向思维=简单原则（避免失败）
- 但执行=不容易（需要自律）

**实践步骤**：
1. **列出失败模式**（10种破产方式）
2. **建立负面清单**（绝不做的事）
3. **设置触发器**（If负面清单，Then拒绝）
4. **定期复盘**（我违反负面清单了吗？）

**优势**：
- 避免大错→长期复利
- 简化决策→减少疲劳
- 心理稳定→持续执行

**下一步**：
- 5月11日：系统思维 - 复杂性与涌现

**Munger名言**：
> "It is remarkable how much long-term advantage people like us have gotten by trying to be consistently not stupid, instead of trying to be very intelligent."
